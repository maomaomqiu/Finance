# M1/M2

M1是什么，M2是什么?

在宏观经济 / 金融 / A股分析里：

## M1 = 狭义货币

- 流通中的现金
- 企业活期存款

👉 代表"可以马上用于交易的钱"

## M2 = 广义货币

- M1
- 居民储蓄存款
- 定期存款

👉 代表"社会总流动性规模"

## 📌 简单理解：

| 指标 | 代表什么 |
|------|----------|
| M1   | 交易活跃度 |
| M2   | 市场流动性总量 |

## 在A股分析里：

- **M1增速上升** → 企业资金活跃 → 股市往往有弹性
- **M2大幅扩张** → 放水 → 资产价格容易上涨

# 缩表

"缩表"是宏观经济里一个非常重要的词，完整叫：

**缩减资产负债表（Balance Sheet Reduction）**

通常指的是 **央行缩表**。

## 🏦 什么是"央行资产负债表"？

以中国为例：

中国人民银行 的资产负债表大概是这样：

**资产端（央行买了什么）**
- 国债
- 政策性金融债
- 外汇资产
- 其他金融资产

**负债端（央行发出去的钱）**
- 商业银行准备金
- 流通中的现金
- 基础货币

## 缩表是什么意思？

👉 **央行减少资产规模**

**怎么做？**
- 不再续买到期国债
- 直接卖出国债
- 收回流动性

**结果就是：**
- 市场上的钱变少 → 流动性收紧

# 扩表

## 第一步：央行买资产

央行：
- 买国债
- 买金融资产
- 向银行投放流动性

👉 资产负债表扩大（扩表）

## 第二步：银行体系钱变多

央行买债 = 把钱打给商业银行

```
银行准备金增加
↓
可放贷资金增加
↓
信贷扩张
```

## 第三步：市场流动性增加

市场上的钱变多：
- 利率下降
- 融资变容易
- 企业资金成本降低

## 第四步：资产价格上涨

资金流向：
- 📈 股市
- 🏠 房地产
- 🪙 大宗商品
- 💻 成长科技股

# 降息(央行)

## 🏦 什么是央行降息？

👉 央行下调基准利率，让整个市场的资金成本变便宜。

**常见的是：**
- 政策利率下调
- 贷款利率下调
- 存款利率下调

**本质一句话：**
> 借钱变便宜

## 📉 降息的资金流动逻辑

```
央行降息
→ 银行融资成本下降
→ 企业/居民贷款利率下降
→ 借钱意愿增强
→ 投资和消费增加
→ 经济刺激
```

## 📈 对市场的影响

### 1️⃣ 对股市
- 贴现率下降 → 股票估值提高
- 融资成本降低 → 企业利润改善
- 资金从存款流向股市

👉 通常利好股市（尤其成长股）

### 2️⃣ 对房地产
- 房贷利率下降
- 月供减少
- 购房需求改善

### 3️⃣ 对债券
- 利率下降 → 旧债价格上涨

# 新门罗主义

**新门罗主义（Neo-Monroeism）**，通常指的是在当代背景下对19世纪美国"门罗主义"的重新解释或强化版本。

## 一、原始的 Monroe Doctrine 是什么？

门罗主义由 James Monroe 于 1823 年提出，核心思想是：

> 欧洲列强不得再干涉或殖民美洲事务，否则将被视为对美国的敌对行为。

**简单理解就是：**
- 🌎 美洲是美国的"势力范围"
- 🚫 欧洲别来插手
- 🇺🇸 美国不干涉欧洲内部事务（当时的表态）

本质是一种**地区排他性的安全主张**。

## 二、什么是"新"门罗主义？

"新门罗主义"不是一个正式政策名称，而是学界和媒体对某些美国外交行为的概括，通常指：

> 在21世纪背景下，美国试图重新强化对西半球（特别是拉美地区）的主导权，排除其他大国（尤其是中国、俄罗斯）的影响。

**它通常包含几个特点：**

### 1️⃣ 排斥外部大国进入美洲

例如：
- 反对 China 在拉美的大规模投资
- 警惕 Russia 在古巴、委内瑞拉的军事存在

### 2️⃣ 强调"后院安全"

美国有时把拉美称为"backyard"（后院），意思是：
- 这里的战略安全必须由美国主导

### 3️⃣ 经济 + 地缘政治结合

比如：
- 供应链重组
- 关键资源（锂矿、能源）
- 港口与电信基础设施控制权

## 三、典型被称为"新门罗主义"的例子

- 在 **Donald Trump 任期内**，美国高官曾公开提到"门罗主义仍然有效"，并在对委内瑞拉、古巴政策上采取强硬立场。
- 在 **Joe Biden 时期**，虽然语气更温和，但在战略层面仍然警惕中国在拉美的影响力。

## 四、和旧门罗主义的区别

| 维度 | 旧门罗主义 | 新门罗主义 |
|------|-----------|-----------|
| 时间 | 1823年 | 21世纪 |
| 对象 | 欧洲殖民国家 | 中国、俄罗斯等全球大国 |
| 背景 | 殖民时代 | 全球化与大国竞争 |
| 手段 | 外交声明为主 | 经济制裁、产业链、军事部署 |
## 五、本质上它是什么？

从国际关系理论角度看，它是一种：
- 🌍 势力范围理论（Sphere of Influence）
- 🛡️ 大国安全缓冲区逻辑
- 🔁 地缘政治回潮

**如果用金融思维理解：**

> 新门罗主义 = 地缘政治版的"区域控盘思维"

就是一个大国希望：
- 控制自己的核心区域
- 阻止其他资本/力量进入
- 保证安全与资源稳定
